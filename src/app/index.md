# `app/`

Next.js App Router pages and configuration.

## Structure

- [layout.tsx](./layout.tsx) — Root server layout with metadata and global providers
- [AboutView.tsx](./AboutView.tsx.md) — About page content rendered by the `/about` route
- [SettingsView.tsx](./SettingsView.tsx.md) — Settings page content rendered by the `/settings` route
- [PrivacyPolicyView.tsx](./PrivacyPolicyView.tsx.md) — Privacy policy content rendered by the `/privacy-policy` route
- [(app-shell)/layout.tsx](./(app-shell)/layout.tsx.md) — Shared client shell for the interactive app routes
- [(app-shell)/page.tsx](./(app-shell)/page.tsx.md) — Lens Editor route (`/`)
- [(app-shell)/glass-map/page.tsx](./(app-shell)/glass-map/page.tsx.md) — Glass Map route (`/glass-map`)
- [(app-shell)/settings/page.tsx](./(app-shell)/settings/page.tsx.md) — Settings route (`/settings`)
- [(app-shell)/privacy-policy/page.tsx](./(app-shell)/privacy-policy/page.tsx.md) — Privacy Policy route (`/privacy-policy`)
- [(app-shell)/about/page.tsx](./(app-shell)/about/page.tsx.md) — About route (`/about`)
