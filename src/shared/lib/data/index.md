# `shared/lib/data/`

Static or shared data sources used across the app.

## Files

- [fraunhoferLines.ts](./fraunhoferLines.ts.md) — Wavelength data for F, d, C Fraunhofer spectral lines
- [exampleSystems.ts](./exampleSystems.ts.md) — Pre-configured example optical systems (e.g., Cooke Triplet)
- [glassCatalogsResource.ts](./glassCatalogsResource.ts.md) — Shared Pyodide-backed glass catalog loader with preload + Suspense read support
