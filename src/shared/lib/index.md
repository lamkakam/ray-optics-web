# `shared/lib/`

Shared domain logic and application utilities. Modules under this directory must not import from `features/`.

## Subdirectories

- [chart-formatting/](./chart-formatting/index.md) — Shared chart value formatting helpers
- [config/](./config/index.md) — App-wide configuration
- [data/](./data/index.md) — Static data constants and resources
- [lens-prescription-grid/](./lens-prescription-grid/index.md) — Shared AG Grid config, cells, and column builders for lens prescription tables
- [schemas/](./schemas/index.md) — Runtime validation schemas
- [types/](./types/index.md) — Core domain TypeScript types
- [utils/](./utils/index.md) — Utility functions and transformations
