# `shared/lib/utils/`

Utility functions and transformations for shared domain logic.

## Files

- [gridTransform.ts](./gridTransform.ts.md) — Transform between Surface data and AG Grid rows (surfacesToGridRows, gridRowsToSurfaces)
- [plotFunctions.ts](./plotFunctions.ts.md) — Functions for creating and displaying optical analysis plots
- [pythonScript.ts](./pythonScript.ts.md) — Utilities for generating Python scripts from optical model state
- [apertureFlag.ts](./apertureFlag.ts.md) — Utilities for handling automatic vs. manual aperture settings
