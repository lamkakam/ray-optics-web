# `components/container/`

State management containers that inject Zustand stores and Pyodide worker into child components via dependency injection.

## Containers

- [LensPrescriptionContainer.tsx](./LensPrescriptionContainer.tsx.md) — Manages lens prescription grid state and surface updates
- [SpecsConfigurerContainer.tsx](./SpecsConfigurerContainer.tsx.md) — Manages optical specs configuration state
- [AnalysisPlotContainer.tsx](./AnalysisPlotContainer.tsx.md) — Manages analysis plot selection and data fetching
- [FocusingContainer.tsx](./FocusingContainer.tsx.md) — Manages focusing algorithm execution and best-focus thickness
- [BottomDrawerContainer.tsx](./BottomDrawerContainer.tsx.md) — Manages bottom drawer open/close state
