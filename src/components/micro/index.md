# `components/micro/`

Minimal, single-responsibility UI primitives. These are base building blocks used throughout the app.

## Components

- [Button.tsx](./Button.tsx.md) — Themed button with variants (primary, secondary, toggle, danger, floating) and sizes
- [Input.tsx](./Input.tsx.md) — Styled text input field
- [Label.tsx](./Label.tsx.md) — Form label for inputs
- [Select.tsx](./Select.tsx.md) — Dropdown select with native or custom options
- [RadioInput.tsx](./RadioInput.tsx.md) — Radio button for mutually exclusive selections
- [Chip.tsx](./Chip.tsx.md) — Small, compact tag/badge component
- [Modal.tsx](./Modal.tsx.md) — Modal dialog backdrop and container
- [Tabs.tsx](./Tabs.tsx.md) — Tab switcher for section navigation
- [Table.tsx](./Table.tsx.md) — HTML table with styled rows and cells
- [Tooltip.tsx](./Tooltip.tsx.md) — Floating tooltip on hover
- [Header.tsx](./Header.tsx.md) — Top navigation header
- [NavLink.tsx](./NavLink.tsx.md) — Navigation link with active state styling
- [Paragraph.tsx](./Paragraph.tsx.md) — Styled text paragraph
- [SetButton.tsx](./SetButton.tsx.md) — Button that sets a store value on click
- [ErrorModal.tsx](./ErrorModal.tsx.md) — Modal for displaying error messages
- [LoadingOverlay.tsx](./LoadingOverlay.tsx.md) — Overlay with loading spinner
- [LoadingMask.tsx](./LoadingMask.tsx.md) — Mask component for loading states
- [AsphericalCell.tsx](./AsphericalCell.tsx.md) — Grid cell editor for aspherical coefficients
- [DecenterCell.tsx](./DecenterCell.tsx.md) — Grid cell editor for decentering values
- [MediumCell.tsx](./MediumCell.tsx.md) — Grid cell editor for selecting optical media
