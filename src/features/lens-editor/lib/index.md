# `features/lens-editor/lib/`

Lens editor feature runtime helpers and constants.

## Files

- [zernikeData.ts](./zernikeData.ts.md) - Zernike ordering constants, index conversion helpers, notation helpers, and classical names.
- [autoSemiDiameters.ts](./autoSemiDiameters.ts.md) - Validates and maps sequential semi-diameters to stable physical surface row IDs.
