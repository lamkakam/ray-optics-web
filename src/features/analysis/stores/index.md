# `features/analysis/stores/`

Zustand stores for the analysis feature.

- [analysisDataStore.ts](./analysisDataStore.ts.md) — Store for computed analysis data (first-order, aberrations, layout image)
- [analysisPlotStore.ts](./analysisPlotStore.ts.md) — Store for analysis plot selection and display state
- [lensLayoutImageStore.ts](./lensLayoutImageStore.ts.md) — Store for cached lens layout plot image
