# `features/analysis/components/DiffractionPsfChart/DiffractionPsfChart.tsx`

## Purpose

Renders the Diffraction PSF analysis view as an Apache ECharts canvas scatter plot. The exported component is generated by `createAnalysisChartComponent(...)`, which owns DOM measurement, responsive sizing, debounced chart updates, and ECharts instance lifecycle management.

## Props

```ts
interface DiffractionPsfChartProps {
  diffractionPsfData: DiffractionPsfData;
  autoHeight?: boolean;
}
```

## Key Behaviors

- Delegates parent measurement, debounce timing, resize handling, and chart disposal to `createAnalysisChartComponent(...)`.
- Uses the parent width as the chart width.
- Injects a sizing policy where `autoHeight` uses a square chart and the fixed-height mode clamps to `min(parentWidth, parentHeight)` while allowing collapse to `0px`.
- Builds chart options through `buildDiffractionPsfOption(...)`.
- Reads the active app theme via `useTheme()` and passes a resolved light/dark chart text color into `buildDiffractionPsfOption(...)`.
- Keeps `data-testid="diffraction-psf-chart"` and `aria-label="Diffraction PSF plot"`.

## Dependencies

- `DiffractionPsfData` from `features/analysis/types/plotData`
- `buildDiffractionPsfOption(...)` from `diffractionPsfChartOption.ts`
- `createAnalysisChartComponent(...)` from `../../lib/createAnalysisChartComponent`
