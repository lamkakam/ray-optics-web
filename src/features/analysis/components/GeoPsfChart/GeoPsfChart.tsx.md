# `features/analysis/components/GeoPsfChart/GeoPsfChart.tsx`

## Purpose

Renders the Geometric PSF analysis view as an Apache ECharts scatter plot. The exported component is generated by `createAnalysisChartComponent(...)`, which owns DOM measurement, responsive sizing, debounced chart updates, and ECharts instance lifecycle management.

## Props

```ts
interface GeoPsfChartProps {
  geoPsfData: GeoPsfData;
  autoHeight?: boolean;
}
```

## Key Behaviors

- Delegates parent measurement, debounce timing, resize handling, and chart disposal to `createAnalysisChartComponent(...)`.
- Uses the parent width as the chart width.
- Injects a sizing policy where `autoHeight` uses a square chart and the fixed-height mode clamps to `min(parentWidth, parentHeight)` while allowing collapse to `0px`.
- Builds chart options through `buildGeoPsfOption(...)`.
- Reads the active app theme via `useTheme()` and passes a resolved light/dark chart text color into `buildGeoPsfOption(...)`.
- Keeps `data-testid="geo-psf-chart"` and `aria-label="Geometric PSF plot"`.
