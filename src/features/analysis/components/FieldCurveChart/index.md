# `features/analysis/components/FieldCurveChart`

Reusable ECharts line-chart component for field curvature and astigmatism curve payloads.

- `FieldCurveChart.tsx` creates the React chart wrapper.
- `fieldCurveChartOption.ts` builds one-grid, one-x-axis, one-category-y-axis options from explicit line-series definitions.
