# `features/analysis/components/spot-diagram-chart/SpotDiagramChart.tsx`

## Purpose

Renders the Spot Diagram analysis view as an Apache ECharts multi-series scatter plot. The exported component is generated by `createAnalysisChartComponent(...)`, which owns DOM measurement, responsive sizing, debounced chart updates, and ECharts instance lifecycle management.

## Props

```ts
interface SpotDiagramChartProps {
  spotDiagramData: SpotDiagramData;
  wavelengthLabels: readonly string[];
  autoHeight?: boolean;
}
```

## Key Behaviors

- Delegates parent measurement, debounce timing, resize handling, and chart disposal to `createAnalysisChartComponent(...)`.
- Uses the parent width as the chart width.
- Injects a sizing policy where `autoHeight` uses a square chart and the fixed-height mode clamps to `min(parentWidth, parentHeight)`.
- Builds chart options through `buildSpotDiagramOption(...)`.
- Reads the active app theme via `useTheme()` and passes a resolved light/dark chart text color into `buildSpotDiagramOption(...)`.
- Keeps `data-testid="spot-diagram-chart"` and `aria-label="Spot diagram plot"`.
