# `features/glass-map/components/`

Components for the interactive glass map (Abbe diagram) feature.

## Components

- [GlassScatterPlot.tsx](./GlassScatterPlot.tsx.md) — Abbe diagram scatter plot with glass selections (visx)
- [GlassDetailPanel.tsx](./GlassDetailPanel.tsx.md) — Displays glass material properties (refractive index, V-number, etc.)
- [GlassMapControls.tsx](./GlassMapControls.tsx.md) — Filter and zoom controls for the glass map view
