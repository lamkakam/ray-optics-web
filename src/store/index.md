# `store/`

Zustand global state stores for app-wide state management.

## Stores

- [lensEditorStore.ts](./lensEditorStore.ts.md) — Main store for lens model state (surfaces, specs, aperture flag)
- [analysisDataStore.ts](./analysisDataStore.ts.md) — Store for computed analysis data (first-order, aberrations, layout image)
- [analysisPlotStore.ts](./analysisPlotStore.ts.md) — Store for analysis plot selection and display state
- [specsConfigurerStore.ts](./specsConfigurerStore.ts.md) — Store for specs configuration dialog state (pupil, field, wavelengths)
- [lensLayoutImageStore.ts](./lensLayoutImageStore.ts.md) — Store for cached lens layout plot image
- [glassMapStore.ts](./glassMapStore.ts.md) — Store for glass map view state (selected glass, filter, zoom)
